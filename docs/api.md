### 신고 요청

- 요청 예시

```json
{
    "id": "게시글 id",
    "신고 사유": "신고 사유에 해당하는 id",
    "디테일 신고 사유": null
}
```

혹은

```json
{
    "id": "게시글 id",
    "신고 사유": "신고 사유에 해당하는 id",
    "디테일 신고 사유": "버스 내부가 너무 더워요"
}
```

- 응답 예시

```
화면에 응답을 뿌리지 않습니다.
응답은 성공, 혹은 실패라는 텍스트만 주시면 됩니다.
```
